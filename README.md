# Python Package Template

This template repository serves as a good starting point for most python packages. Features include:

  * github action for automatically running unittests
  * github action for automatically building docs using sphinx

## Running Tests

Tests are built and run using [pytest](https://docs.pytest.org/en/7.1.x/). To run tests, run the following:

```sh
$ pytest -q tests/
```

So long as all files in the test directory are named with convention
